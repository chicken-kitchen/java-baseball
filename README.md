# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
* 게임 준비
    * 정답 숫자 생성
        * 1부터 9까지 서로 다른 수
* 게임 진행 (3 스트라이크까지 반복)
    * 인입 숫자 생성
        * 1부터 9까지 서로 다른 수
    * 결과 판단
        * 스트라이크 - 같은 수, 같은 자리
        * 볼 - 같은 수, 다른 자리
        * 낫싱 - 다른 수
    * 결과 출력
* 게임 종료
    * 재시작 및 종료
        * 1: 재시작
        * 2: 종료
