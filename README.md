# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
* 3자리 수 생성
    * 1부터 9까지 서로 다른 수
* 숫자 입력
    * 1부터 9까지 서로 다른 수
* 판단
    * 스트라이크 판단
        * 같은 자리, 같은 수
    * 볼 판단
        * 다른 자리, 같은 수
    * 낫싱 판단
        * 모두 다른 수
* 결과 출력
* 게임 반복 및 종료
    * 3스트라이크면 종료
* 재시작 및 종료
    * 1: 재시작
    * 2: 종료
